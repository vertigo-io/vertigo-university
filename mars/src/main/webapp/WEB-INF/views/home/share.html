OpenData

The MMC Command Center publishes Open Data sets about bases, equipment units and equipment types, and teams.

The Mars Mining Corporation leverages the Open Data Soft service to make these data sets available to its business partners and also to terran researchers.

Api

API Management permit to open the information system to a targetted audience.

With digital transformation API are now largely used and standardized. To deal with the proliferation of APIs, API Management provide API stores that catalog all accessibles APIs for use.
With API Management, consummers can now freely browse and register to use those API by using securized access.
On the usage, quota alongside with billing of these API can now be used directly without the need to develop a custom solution.
Lifecycle management is also handled by the product: versionning and status can now be operated throught the administrative part of this solution.
Moreover, this solution records usage statistics and analytics and a dashboard is used to provide control and performance indicators of published API and technical alerts.

  • Catalog existing API
  • Freely browse and register API with securized access
  • Quota and billing management
  • Easy Lifecycle Management
  • Indicators with dashboard
  • Alerts
